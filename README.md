# Learn React Redux Toolkit

Hands-on notes, guides, and examples for learning Redux Toolkit with React. Built on Vite for a fast dev experience, with a curated set of markdown docs you can follow in sequence.

---

## Quick Start

Install dependencies and run the dev server:

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```

## Features

- Vite + React with HMR and ESLint
- Clear Redux Toolkit how-tos and patterns
- Async flows with `createAsyncThunk`
- Normalized state with Entity Adapter
- Selectors and memoization with Reselect
- RTK Query overview for server state
